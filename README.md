# SPK Tech Wiki
Welcome to the SPK Tech Wiki! This is your go-to resource for technical documentation, guides and basic event setups.

# Contributing
To contribute to this wiki, click on 'edit this page' at the bottom of any page. You can create pull requests with your changes, and they will be reviewed before being merged. If you can't seem to proceed with your changes, feel free to reach me out at hi@seneth.me.
