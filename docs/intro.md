# Introduction

In this documentation, you will find all the necessary information to get started the equipments, and how to set the up for different events.

You do not need prior experience. You do need to read.

## How to use this wiki

- Use the Sidebar to browse by category.
- Use the search bar if you know what you are looking for.
- Each equipment has its own page.
- Available manuals and guides are linked on each equipment page.
